std::string ArchiverApplication::getUsage ( ) const
inlineprivatevirtual

Overload this method to describe the usage of the application and to give a (short) description of the application.

Default return string is "usage: " + getBaseName() + "[OPTION(S)]\n" Return the usage of the application.

Normally, that method does not have to be overloaded; overload getApplicationDescription() and getParamterDescription() and getDefaultParameterDescription() instead.
