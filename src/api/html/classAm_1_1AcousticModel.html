Detailed Description

Interface class for acoustic models Concepts and Future Plans: -An acoustic model consist of: -Emission probability set.

Interface: Mm::FeatureScorer -Tying of allophone states concerning the emission probabilities. Interface: emissionIndex(...) -Transition probability set (only preliminary implementation) -Tying of transition probability concerning transitions. (Not implemented yet.) -HMM topology. Preliminary interface: hmmTopology(...) Further functionalities -Create allophone state to phoneme transducer. Interface createAllophoneStateToPhonemeTransducer -Deliver the phoneme inventory. Interface phonemeInventory(). -Deliver the allophone state alphabet. Interface allophoneStateAlphabet(). -Deliver a phonology object. (Needs refinements.) Interface phonology(). -Deliver the silence identifier. (Not sure if necessary.) Interface silence(). -Delivers if across-word model is enabled. (Not sure if necessary.) Interface isAcrossWordModelEnabled().
