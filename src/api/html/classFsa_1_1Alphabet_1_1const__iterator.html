Detailed Description

Iterator that should be used to traverse an alphabet.

If you derive from Alphabet to create your own implementation of an on-demand alphabet you should NOT modify this iterator. Instead overload appropriate virtual methods at your Alphabet class. You should not assume any ordering on the label ids or strings you get.
