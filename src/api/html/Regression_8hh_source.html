25 /**
26  * Calculation of derivatives with linear regression coefficients.
27  *
28  * Here only the 1-dim. case for the second derivative is described.
29  * All dimensions are handled independently.
30  *
31  * Given a symmetric window of e.g. +/- 2 frames, the regression
32  * data can be defined as
33  *
34  * x = [ -2, -1, 0, 1, 2 ]^T
35  * f = [f(-2), f(-1), f(0), f(1), f(2)]^T where f is a feature
36  *
37  * Assume now the vector y is generated by a polynomial
38  *
39  * y = b0 + b1*x + b2*x^2
40  * y" = 2*b2
41  *
42  * or, equivalently y = X*B with
43  *
44  * X = [x.^0, x.^1, x.^2] -> 5x3 input matrix
45  * B = [ b0, b1, b2]^T -> 3x1 coefficients
46  *
47  * With least squares, we obtain
48  *
49  * B = (X^T*X)^-1*X^T * f
50  * = C * f
51  * (3x5) * (5x1) = (3x1) dim
52  *
53  * Thus the coefficient b2 is the last row of B and the second derivative
54  * is approximated with the last row of C by
55  *
56  * y" = 2 * b2
57  * ~ 2 * (C(end,:) * f)
58  *
59  * For the second derivative, the coefficients 2*C(end,:) result in
60  *
61  * [2 -1 -2 -1 2]/7
62  *
63  * for the first derivative 1*C(end,:) = b1 (same idea, but start
64  * with y = b0 + b1*x):
65  *
66  * [-2 -1 0 -1 2]/10
67  *
68  * With some algebra, the derivatives can be calculated directly
69  * by evaluating:
70  *
71  * y' = (sum_i=1^2{ i*( f(i) - f(-i)) }) / (2*sum_i=1^2 i^2)
72  * y" = (sum_i=1^2{ i*( y'(i) - y'(-i)) }) / (2*sum_i=1^2 i^2)
73  */
