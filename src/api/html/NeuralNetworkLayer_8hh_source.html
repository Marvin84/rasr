36 /** Abstract base class for all neural network layers
37  *
38  * To create a new neural network layer, derive your new layer from this
39  * base class, add a new layer type, and extend the "createNeuralNetworkLayer"
40  * method.
41  *
42  * Current neural network layer implementations
43  * - activation functions
44  * - identity
45  * - sigmoid
46  * - softmax
47  * - tanh
48  * - rectified
49  * - linear (feed-forward)
50  * - combined: parameter + activation
51  * - linear+sigmoid
52  * - linear+softmax
53  * - linear+rlu
54  * - linear+tanh
55  * - pre-/post-processing steps
56  * - logarithm
57  * - PCA (mean + variance)
58  * - polynomial
59  * - GaussianNoise
60  *
61  * The layer can have multiple inputs from other layers.
62  * Inputs are connected at "ports", which are just indices of the connections specified in the configuration.
63  * The layer has only one output, which also has a "port" index.
64  * When building up the network, the predecessor layers need to be set, because they are required in backpropagation.
65  * Plans: Simplify design of neural network layer, remove redundancy
66  *
67  */
