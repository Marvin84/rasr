67 /**
68  * automaton properties
69  *
70  * storage:
71  * - automaton stores all states and arcs (instead of calculating them)
72  *
73  * linear:
74  * - outdegree (and therefore indegree) of all states is 1
75  *
76  * sausages:
77  * - automaton has an initial and a single final state
78  * - indegree and outdegree of all states except initial and final is 1
79  *
80  * acyclic:
81  * - automaton does not contain any cycle except the trivial ones
82  *
83  * cached:
84  * - automaton is being cached; helps to avoid multiple caching
85  * (we will use a special class field for this in the future)
