Detailed Description

template<class A>
class OpenFst::CompactReplaceFst< A >

A simple and compact ReplaceFst.

Adapted from OpenFst ReplaceFst. Recursive replacements are not supported. All nonterminals have to be negative. At most 255 nonterminals are support. Nonterminals should be dense.
