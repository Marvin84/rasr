Detailed Description

Interface for C o L transducer.

The actual type of the underlying transducer depends on the type of label/weight look-ahead and the accumulator used for weight pushing. All of the parameters are template arguments in OpenFst, therefore a polymorphic wrapper is required.

The derived classes take care of loading or creating a MatcherFst, relabeling the G transducer (if required) and creating the ComposeFst (using ComposeFstFactory)
