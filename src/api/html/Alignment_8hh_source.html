50 /*
51  * Alignments are usually stored as a sequence of allophone state ids.
52  * This is useful, because it makes the alignment independent of the state tying.
53  * For some purposes, the state tying can be assumed to be fix. Mapping the alignment
54  * to the emission label can cause some overhead with our implementation.
55  * In particular, reading the lattice-alignment
56  * in sequence-discriminative training is expensive.
57  * Therefore, we also allow writing the emission indices directly.
58  */
