29 /**
30  * Epsilon Closure Filter:
31  *
32  * Given states s and e. Pathes_w(s,e) is the set of all pathes from
33  * s to e having exactly one arc labeled with w and all others labeled
34  * with epsilon. Arcs_w(s,e) is the set of all arcs in Pathes_w(s,e)
35  * labeled with w. Arcs_s'/w(s,e) is the set of all arcs in Arcs_w(s,e)
36  * having source state s'. Pathes_s'/w(s,e) is the subset of Pathes_w(s,e),
37  * such that each path in Pathes_s'/w(s,e) includes an arc in Arcs_s'/w(s,e).
38  *
39  * for each w, (s,e):
40  * 1) for each a in Arcs_w(s,e) keep only the best scoring path in Pathes_w(s,e)
41  * that includes a.
42  * -> classical epsilon-removal over the tropical semiring
43  * 2) for each s' keep only the best scoring path in Pathes_s'/w(s,e)
44  * -> classical epsilon-removal over the tropical semiring
45  * with statewise determinization
46  * 3) keep only the best scoring path in Pathes_w(s,e)
47  * -> classical epsilon-removal over the tropical semiring
48  * with determinization over all pathes from s to e;
49  * attention:
50  * under some circumstances the determinization is not given;
51  * the problem could be avoided by introducing new states and arcs,
52  * but that could result in a (huge???) blow up of the lattice size
53  *
54  * Arcs and states are removed, but never new states and arcs are intorduced.
55  * The implementation is static, i.e not lazy.
56  **/
57 StaticLatticeRef applyEpsClosureFilter(ConstLatticeRef l);

59 StaticLatticeRef applyEpsClosureStrongDeterminizationFilter(ConstLatticeRef l);
