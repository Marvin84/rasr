Detailed Description

Base class for Flow packets with time stamp information.

The time stamp indicates the perdiod of time in the original recording corresponding to the data packet. Time values are measured in seconds from the beginning of the recording. start() and end() are understood to denote the left-close, right-open intervall [start, end). This means that in a stream of contiguous, non-overlapping packet, the end time of a packet is equal to the start time of the next packet.
