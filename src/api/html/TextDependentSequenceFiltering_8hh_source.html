26 /**
27  * Converts alignment into sequence selection and/or calculates ratio of the not-filtered timeframes.
28  * Input:
29  * default port: alignment
30  * Outputs:
31  * default port: vector-bool i.e. selection of remaining timeframes
32  * ratio port: f32, i.e. #remaining / #all timeframes.
33  *
34  * -Selection set types:
35  * -set of central phonemes; set name 'phoneme-selection'.
36  * -set of state identifiers (tipically 0, 1, 2); set name 'state-selection'
37  * -Parametrization of selection sets:
38  * -parameter 'symbols' is list of symbols inlcuded in the set (e.g. in configuration files:
39  * phoneme-selection.symbols=a v f; in flow files: phoneme-selection.symbols="a v f")
40  * -parameter 'complement': if true the symbol set is complemented (e.g. in configuration files:
41  * state-selection.complement=true; in flow files: state-selection.complement="true");
42  * Note: for empty sets, 'complement' is set automatically to 'true'.
43  * -Selection sets are in 'AND' relation, thus those timeframes are selected whose allophone
44  * states are included in all sets.
45  * -Sequence selection object is a vector of bool values. Timeframes not included in one of
46  * the selection sets are marked by false and timeframes included in all selection sets are marked by true.
47  * @see Flow::SequenceFilterNode for further explanations.
48  */
