bool Alignment::setAlphabet ( Core::Ref< const Fsa::Alphabet >  alphabet,
bool  skipMismatch = false 
)

Set a mapping alphabet.

When this is set, the alphabet is used to robustly map allophone indices between different alphabets. If skipMismatch is true, then alignment items which could not be mapped into the new alphabet are simply removed. Otherwise an error is raised for such items.
