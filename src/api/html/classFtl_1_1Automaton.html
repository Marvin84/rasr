Detailed Description

template<class _Semiring, class _State = State<Arc<typename _Semiring::Weight>>>
class Ftl::Automaton< _Semiring, _State >

The basic automaton data structure.

Automaton is a virtual base class to ALL other automaton classes. It defines the abstract, lazy interface to access an automaton. All algorithms are based only on this abstract definition. Lazy automaton classes therefore derive from this class and overload the appropriate methods.

template<class _Semiring, class _State = State<Arc<typename _Semiring::Weight>>>
virtual ConstStateRef Ftl::Automaton< _Semiring, _State >::getState ( Fsa::StateId  s) const
pure virtual

Returns a reference to a state.

All algorithms assume that this does not fail (return an empty reference) if the state id has been reported by the abstract automaton itself before (either through initialStateId() or target state ids on arcs).

Warning
Note that in case of lazy algorithms calling this method may take some time as states, i.e. mainly the list of outgoing arcs, are calculated on-demand.
Parameters
sid of the requested state
Returns
return a reference to the state with id s or an empty reference if the state does not exist
