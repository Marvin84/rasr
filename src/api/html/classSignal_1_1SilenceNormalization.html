Detailed Description

This node normalizes the fraction of silence in the outgoing signal.

The returned speech segments contain exactly the requested amount of silence.

An EM-like algorithm is used to segment the silence from the non-silence, which expects that there is both silence and speech in the segment. Therefore data put into this node should already have been somehow segmented externally.

The constraints are: At least 1% of the signal must be silence, and at least 1% must be speech (see the parameter initialization-fraction).
