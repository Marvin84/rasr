Detailed Description

Interface for performing a search for the best string in an Fsa w.r.t.

minimum Bayes risk. This kind of search takes only an Fsa and a configuration as an argument. For statistics purposes, the number of evaluated hypotheses and the number of computations performed during the search can be optionally computed by the implementing classes.
