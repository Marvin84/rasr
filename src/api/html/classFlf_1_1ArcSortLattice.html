Detailed Description

template<class WeakOrder>
class Flf::ArcSortLattice< WeakOrder >

Filter: true, if arc has to be removed.

WeakOrder: order arcs per state w.r.t. to matcher, postcondition: for each state holds that a non-matching arc is never followed by a matching arc
