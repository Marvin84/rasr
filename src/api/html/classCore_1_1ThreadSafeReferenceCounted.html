Detailed Description

Base class for reference-counted objects used in a multithreaded environment.

To enable reference counting for a class it need to inhert this class publically.

Remark: ThreadSafeReferenceCounted is by intention not derived from ReferenceCounted to forbit access of thread safe reference count by a not thread safe Core::Ref object.
