Detailed Description

Implementation of the naive approach for minimum Bayes risk using an nbestlist for both the evaluation space as well as for the summation space.

The search can be performed on two different nbestlists depending on how much hypotheses shall be used for minimization or summation. In order to stay conform to the MinimumBayesRiskSearch interface, the search itself will be given a single nbestlist from which the two lists for evaluation space and summation space will be extracted up to a configurable number of string hypotheses. In addition there exist two criteria which can abbreviate the search process. Their application can be configured. If fulfilled, this information will be logged to the information channel. It should be noted that the computation of the two criteria do not afford additional computation time but come along with the computation process.
