Detailed Description

template<class T>
class Nn::BufferedAlignedFeatureProcessor< T >

The class combines the "BufferedFeatureExtractor" and the "AlignedFeatureProcessor" and provides the buffered processing of the features and alignment .

This should be used by Speech::AligningFeatureExtractor.
