Detailed Description

template<class T>
struct Core::CompressedPlainTextFormat< T >

Template helper class for registering plain text formats.

To register plain text format for class Foo in a format set call the function Formatset::registerFormat("ascii", new Core::PlainTextFormat<Foo>());
