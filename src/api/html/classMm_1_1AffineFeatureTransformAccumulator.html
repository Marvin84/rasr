Detailed Description

AffineFeatureTransformAccumulator.

Implementation of C-MLLR or Feature Space MLLR, as described in [Gales 1998] Extended to handle dimension reducing transformations, by using the degenerate MMI criterion [Visweswariah 2004]. The extension is equivalent to ML estimation for non-projecting transformations.
