Fsa::ConstAutomatonRef Aligner::getAlignmentFsa ( ) const

Returns an automaton representing the alignment of HMM model states and observations (features).

In the case of Viterbi training, the resulting acceptor has a simple linear structure representing the best path. With Baum-Welch Training, we have an acyclic graph representing the search space of the aligner limited to those paths which reached a final state. In either case, an allophone-state acceptor with arc weights being acoustic scores is returned. If the aligner does not reach a final state due pruning settings which are too restrictive, an empty automaton is returned.
