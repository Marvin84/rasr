27 /** Implementation of bayes decision rule for "simple" distributions.
28  *
29  * Provides the class k with maximum a-posterioiry probability
30  * i.e minimum with -log(a-posterioiry probability):
31  *
32  * argmin_{k} -log( p(k) * p(x|k) )
33  * where p(k) is the apriori probablity of the classes and
