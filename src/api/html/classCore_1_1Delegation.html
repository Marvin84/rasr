Detailed Description

template<typename T>
class Core::Delegation< T >

Implements a kind of the delegation pattern.

Use set to register either an instance of Delegation::Target or a member function of type "void C::*(T)", where C is an arbitrary class. A call to delegate will now delegate the call to the registered target.

Attention: If a target is registered or not has to be checked externally; do not assume that delegate do this check.
