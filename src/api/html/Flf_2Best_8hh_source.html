23 /**
24  * Dijkstra:
25  * - aka Ftl::firstBest
26  * - extend path scores, find path with minimum w.r.t. to projection of extended path scores
27  * - uses priority queue to follow currently best path
28  * Bellman-Ford:
29  * - aka Ftl::best
30  * - extend path scores, collect extended path scores
31  * - general single source shortest path (SSSP) algorithm
32  * Projecting-Bellman-Ford:
33  * - project scores, sum projected scores, find path with minimum sum of projected path scores
34  * For the tropical semiring and if the projection of each score is positive,
35  * then all three algorithms yield the same result.
36  **/
