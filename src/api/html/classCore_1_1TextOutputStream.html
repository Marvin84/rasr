Detailed Description

Stream for text output.

Use this for writing plain text files. For XML use XmlOutoutStream.

Features if TextOutputStream:

  • word wrap
  • indentation
  • character set conversion Remeber: all internal string are considered UTF-8

TextOutputStream redefines some ASCII control characters to control text formatting. Since these codes are old teletype legacy and have no use in modern computing environments, we should get away with this.
