Detailed Description

A WFST-based decoder.

Supports dynamic expansion of HMM states (therefore "Expanding"). The WFST used as search network can be constructed statically or dynamically. This class serves as interface to the Speech::Recognizer object and handles the parameters. The main work is performed in SearchSpace.
