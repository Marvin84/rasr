Detailed Description

Cache for Fsa object.

Cache stores FSA objects in a archive. Use 'find', to get an existing object. Use 'insert' to add a new object. Use 'get', to retrieve object which will be created if not existent.

Remark: This is a simple implementation, once Core::ObjectCache supports reading of Core::Ref, this implementation could be easly merged with it.
