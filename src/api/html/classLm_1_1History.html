std::string Lm::History::format ( ) const

Create string representation (for debugging puposes).

The preferred way to access this function is through calling LanguageModel::formatHistory(), since some language models specialize this formatHistory() to give more readable information. If your language model is m-gram like, describe the history chronologically, i.e. most recent word last.
