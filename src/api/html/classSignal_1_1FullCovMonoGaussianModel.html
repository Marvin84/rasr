const f32 FullCovMonoGaussianModel::computeComplexity ( )

BIC is a likelihood criterion penalized by the model complexity: the number of parameters in the model.

The number of parameters for each cluster is d+d/2(d+1)

with penalty function P = (1/2(d+d/2(d+1))) * log(N) = (2d/4 + d^2/4 + d/4) * log(N) = (1/4(d*(3+d))) * log(N)
