33 /** Interface class for acoustic models
34  * Concepts and Future Plans:
35  * -An acoustic model consist of:
36  * -Emission probability set. Interface: Mm::FeatureScorer
37  * -Tying of allophone states concerning the emission probabilities. Interface: emissionIndex(...)
38  * -Transition probability set (only preliminary implementation)
39  * -Tying of transition probability concerning transitions. (Not implemented yet.)
40  * -HMM topology. Preliminary interface: hmmTopology(...)
41  * Further functionalities
42  * -Create allophone state to phoneme transducer.
43  * Interface createAllophoneStateToPhonemeTransducer
44  * -Deliver the phoneme inventory. Interface phonemeInventory().
45  * -Deliver the allophone state alphabet. Interface allophoneStateAlphabet().
46  * -Deliver a phonology object. (Needs refinements.) Interface phonology().
47  * -Deliver the silence identifier. (Not sure if necessary.) Interface silence().
48  * -Delivers if across-word model is enabled. (Not sure if necessary.)
49  * Interface isAcrossWordModelEnabled().
50  */
