Detailed Description

Implementation of bayes decision rule for "simple" distributions.

Provides the class k with maximum a-posterioiry probability i.e minimum with -log(a-posterioiry probability):

argmin_{k} -log( p(k) * p(x|k) ) where p(k) is the apriori probablity of the classes and p(x|k) is the class dependent likelihood function.
