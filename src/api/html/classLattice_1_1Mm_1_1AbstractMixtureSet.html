Detailed Description

AbstractMixtureSet administrates mixture objects and their relations.

Data structure: -objects pointers are stored in tables; -objects points reference others by integer indexes. Advantage of pointer representation: -object types can be extended be deriving from the base types and without changing (much :-) ) in other parts of this library Advantages to graph representation: -direct access to set of elements (e.g.: list of all covariances) -storing and load is straight forward Disadvantages: -hard to remove elements from the structure -since the tables contain pointers, each access needs +1 indirection. Thus this representation is not suitable for fast computations.
