Enumeration Type Documentation

enum utf8::ByteType

UTF-8 is based on bytes (8 bit).

Codepoints below 128 (i.e. ASCII characters) are represented as a singleByte. Other codepoints are encoded as multi-byte sequence, which consists of a multiByteHead followed by one up to five multiByteTail bytes. The two byte values 0xfe and 0xff are illegal, they can never occur in a UTF-8 string.
