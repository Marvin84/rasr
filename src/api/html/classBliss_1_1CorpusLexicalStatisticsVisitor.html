void CorpusLexicalStatisticsVisitor::visitSpeechSegment ( SpeechSegment *  s)
virtual

Template method for processing speech segments.

Implement this to process the speech segments of a corpus. By default the request will be forwarded to visitSegment(), so ignore this if you want to treat speech and non-speech segments alike.

Reimplemented from Bliss::SegmentVisitor.
