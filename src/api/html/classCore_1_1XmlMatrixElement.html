Detailed Description

template<class T>
class Core::XmlMatrixElement< T >

XmlMatrixElement implements XmlBuilderElement for an arbitrary XmlSchemaParser which parses a matrix of a given type.

Matrix object is of type Math::Matrix<T> XML tag name: matrix-type. E.g.: matrix-f32

Elements should be stored row after row, separated by whitespace characters. Remark: end-of-line character does not have any special role.
