f32 Bliss::SyntacticToken::classEmissionScore ( ) const
inline

Probability of a lemma with this syntactic token.

Several lemmata can be assigned to the same syntactic token. Conceptually we can write this as p(lemma | history) = p(lemma | synt) p(synt | history). This function returns only - log(p(lemma | synt)) under the assumption of a unifor distribution.

Warning
The interaction with language model phrases is a theoretically unsolved problem.
