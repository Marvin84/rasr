51 /**
52  * Phonology for context dependent phonemes.
53  *
54  * The @c ContextPhonology allophone is the @c PhonemeInContext.
55  * A phoneme in context is defined by its underlying phoneme and
56  * its left and right context phonemes. Such units are commonly
57  * referred to as "n-phones" (e.g. triphone, quinphone,...).
58  *
59  * The conversion from a phoneme graph to a context-dependent
60  * phoneme graph involves replication of vertices and edges
61  * according to the various contexts in which they occur. The
62  * current implementation proceeds as follows:
63  * -# Determine all left contexts (histories) of each vertex.
64  * -# Determine all right contexts (futures) of each vertex.
65  * -# For each phoneme graph vertex build the cartesian product of
66  * left and right contexts and create an allophone graph vertex
67  * for each context.
68  */
69
