Detailed Description

Base class for normalization algorithms.

Processing: updateStatistics is called if no new data entered or old data left the sliding window finalize() called right before normalization is applied to an output vector apply() called to perform normalization Interface: Use update(..) to put new data into the sliding window and get the latest normalized output Use flush to get normalized output without any input
