Detailed Description

template<typename T>
class Nn::RpropEstimator< T >

Implementation of the Rprop algorithm of Riedmiller & Braun.

The "backtracking" of the original Rprop algorithm is not implemented, because it is known to be harmful.

The implementation is not intended to be used on GPU. In GPU mode, it uses a lot of sychronization and computation on CPU.
