Detailed Description

template<class O>
struct Search::Wfst::ComposeFstFactory::Compose< O >

Construction helper.

Generates ComposeFstImplOptions and call the ComposeFst constructor. create() is overloaded in order to get the correct Fst for DynamicLmFst (see ComposeOption above).
