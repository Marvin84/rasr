Lattice::ConstWordLatticeRef Search::Aligner::WordLatticeBuilder::build ( Fsa::ConstAutomatonRef  alignmentFsa)

Builds the word lattice for a given traceback and model-transducer.

ModelTransducer (

See also
setModelTransducer) is used to find the word (e.i. lemma-pronunciations) boundaries. It is composed by a allophone state acceptor created from the aligment. The result of the composition contains word boundary (i.e. disambiguator) arcs and lemma-pronunciation output labels. The shortest path of the composition gives finally to necesary informations (e.i. allophone states, lemma-pronunciations, and acoustic scores) to create a linear word lattice from an alignment traceback.
