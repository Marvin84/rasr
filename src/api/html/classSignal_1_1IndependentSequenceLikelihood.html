Detailed Description

IndependentSequenceLikelihood implements the class dependent density of feature vector sequence: sum_{t} ( - log p(x_t | k) )

Assumptions: (i) feature vectors are independent (ii) classes are independent

Density p(x_t | k) is dependent on the FeatureScorer object.
