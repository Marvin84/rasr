Detailed Description

Choice class.

A choice is the symbolic mapping between enumerations and strings. Choices are used to specify enumerative parameters.

Constructs the choice.

The parameters are specified as a list of pairs of strings and enumeration (integer) values. varargs are used to allow for an arbitrary number of different choices. The last identifier must be 0 to mark the end of the list.
