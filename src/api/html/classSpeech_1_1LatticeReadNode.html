bool LatticeReadNode::work ( Flow::PortId  out)
virtual

Processes some data.

Implement this function to do whatever your node is supposed to. You are required to produce at least one packet of data on the specified output out. You may send more packets, also to other outputs.

Returns
true on success, false on end-of-file or error. The idiom is:
return putData(out, d);
