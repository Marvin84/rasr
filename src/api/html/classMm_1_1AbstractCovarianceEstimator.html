virtual Covariance* Mm::AbstractCovarianceEstimator::estimate ( const CovarianceToMeanSetMap &  map,
VarianceType  minimumVariance 
)
inlinevirtual

Overload this function to perform covariance estimation.

The map contains mean estimator references for each covariance estimator. The minimumVariance specifies the minimum value in covariance matrix. If minimum variance is not supported by a derived estimator class, overload the protected estimate function without this additional parameter.

Reimplemented in Mm::CovarianceEstimator, Mm::RpropDiscriminativeCovarianceEstimator, and Mm::EbwDiscriminativeCovarianceEstimator.
