Some criterions will calculate some kind of pseudo targets and have the gradient -{y} / y w.r.t.

y, and when y = softmax(a), they have the gradient y - {y} w.r.t. a. In that case, we call {y} the pseudo targets. Examples: Cross-Entropy and CTC. Can return NULL. Otherwise the pointer will be valid until the next input.
