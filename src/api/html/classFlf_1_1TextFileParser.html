Detailed Description

Parse Textfile line by line and split at (sequences of) white spaces into columns; empty lines and lines containing a comment only are discarded.

Supports:

  • Encoding -> see Core::TextInputStream
  • Escaping -> if <c> occurs, is appended to the current column
  • Quoting -> Columns may be bracketed by "; escaping is active within quoting, comments are not
  • Comments -> Text starting with # or ;; is discarded until the eol

Example: 1: a\ b# "a b#" -> ["a b#", "a b#"] 2: a\ b# "a b#" -> ["a b"] 3: "a b#" a\ b# -> ["a b#", "a b"]
