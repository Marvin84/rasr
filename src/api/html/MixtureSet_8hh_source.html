29 /**
30  * AbstractMixtureSet administrates mixture objects and
31  * their relations.
32  *
33  * Data structure:
34  * -objects pointers are stored in tables;
35  * -objects points reference others by integer indexes.
36  * Advantage of pointer representation:
37  * -object types can be extended be deriving from the base types and
38  * without changing (much :-) ) in other parts of this library
39  * Advantages to graph representation:
40  * -direct access to set of elements (e.g.: list of all covariances)
41  * -storing and load is straight forward
42  * Disadvantages:
43  * -hard to remove elements from the structure
44  * -since the tables contain pointers, each access needs +1 indirection.
45  * Thus this representation is not suitable for fast computations.
46  */
