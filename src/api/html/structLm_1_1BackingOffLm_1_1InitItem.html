Detailed Description

Back-off language model item.

For items with non-zero token, score gives the negative natural logarithm of p(token | history): score = - ln ( p(token | history) )

Items with token equals zero provide back-off weights: score = - ln ( p(w | history) / p(w | shortened_history) ) Obviously this term has to be constant over all words w for which the back-off case occurs. (Actually, this could be used as the the definition of "back-off case".)
