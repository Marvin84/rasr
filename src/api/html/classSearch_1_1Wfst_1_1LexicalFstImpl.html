Detailed Description

template<class M>
class Search::Wfst::LexicalFstImpl< M >

Manages loading, converting, and deleting of the underlying fst.

The LexicalFstImpl object is stored in the AbstractLexicalFst, which forwards the calls to load() and create(). fst_ is a pointer to the parent's member, such that the parent can use the correct pointer type without casting.
