Detailed Description

Combination of a lexicon, an acoustic model, and a language model.

It supports creation and initialization of these three mutually dependent objects.

Usage: 1) create ModelCombination object locally to create the three parts: lexicon, acoustic model, and language model.Store references of those parts which you 2) call function load, to load the scaling values 3) Store the references to those parts you will use later. 4) When the local ModelCombination object get destructed, the unreferenced parts gets freed as well.
