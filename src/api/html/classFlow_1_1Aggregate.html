Detailed Description

Aggregate of Flow objects Aggregate collects objects of different streams into one object.

It supports polymorfism: -The base class for the input objects is Timestamp. Thus, input objects must be instances of the base class or of one of its derived classes. -I/O is solved using features supported by Data.
