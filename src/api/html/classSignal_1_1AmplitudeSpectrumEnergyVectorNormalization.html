Detailed Description

template<class T>
class Signal::AmplitudeSpectrumEnergyVectorNormalization< T >

AmplitudeSpectrumEnergyVectorNormalization: divides each element by square of frame energy calculated form (half) amplitude spectrum.

square of frame energy = ( 1 / N * (2 * sum_{omega=1}^{N/2 - 1} (x^2(omega)) + x^2(0) + x^2(N/2)) ) ^{1/2} where N is length of FFT

Applied normally to amplitude like values.
