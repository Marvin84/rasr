Detailed Description

template<class T, class Parser>
struct Core::XmlFormat< T, Parser >

Template helper class for registering xml formats.

To register xml format for class Foo in a format set call the function Formatset::registerFormat("xml", new Core::XmlFormat<Foo, FooParser>());

Template parameter Parser is a class, which can parse an XML document and initialize an object of type T.
