152 /*
153  * A Word Lattice.
154  *
155  * A set (vector) of fsas All fsas are expected to be isomorphic,
156  * i.e. they have the same state numbering, same arcs and same
157  * lables; only the weights may differ. The fsa are
158  * LemmaPronunciation acceptors. Word boundary information is
159  * assciated with the states of the FSAs via the state id.
160  *
161  * Note: Previosly the word lattice was a LemmaPronunciation to
162  * SyntacticToken transducer. Although this was more consistent
163  * with state-machine orthodoxy, the SyntacticToken labels have
164  * proved to be not only redundant, but actually useless in
165  * practice. A convenience slave adaptor could be provided to
166  * restore them. This should then include the correct
167  * pronunciation and syntactic class weights. (Since these had
168  * been missing, the SyntacticToken labels were useless.)
169  *
170  * Plans: In the future we want to to represent the set of FSA
171  * automata by weight multiplexing: We use a single FSA with a
172  * dummy semiring. The weights of the automaton are actually
173  * indices enumerating the arcs. The different weight assignments
174  * are represented by a set of vectors. FSA adaptors (derived
175  * from ModifyAutomaton) will be provided to emulate a normal
176  * Fsa::Automaton with the selected weights.
177  */
