Detailed Description

template<class A>
class OpenFst::EpsilonEncodeMapper< A >

Behaves like EncodeMapper, except for epsilon arcs with trivial weight, which are always mapped to label 0.

Due to the lack of virtual functions in EncodeMapper, the additional logic is added using the Decorator Pattern.
