Detailed Description

template<class ArcScores>
class Flf::FramewiseCollector< ArcScores >

Calculate time frame wise word posterior distribution from arc confidence score, i.e.

p(w|t) = (sum_{e E: begin(e) <= t < end(e) and word(e) = w} conf(e)) / (sum_{e E: begin(e) <= t < end(e)} conf(e))

The result is a list of posterior distributions p(*|t) for t = 1,...,T
