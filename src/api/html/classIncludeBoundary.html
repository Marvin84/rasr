Detailed Description

Include Boundary: left and right boundaries of input are included but with small weights.

-First filter starts at center equals spacing. -Last filter is chosen such that its left border is smaller than maximal-frequency and its right border is larger or equal to the maximal-frequency. =>Number of filters = ceil(maximum-frequency - (1.0 - normalized-center) * filter-width) / spacing) -Sample rate of output vector: 1.0 / spacing. -Note that if spacing > normalized-center * filter-width, the leftmost few input values will not contribute to the results.
