Detailed Description

Traverse a lattice.

traverseDfs: Traverse lattice using the dfs algorithm. traverseInTopologicalOrder: Traverse lattice in topological order; lattice has to be acyclic. traverse: Use fastes way to traverse lattice, i.e. if pre-calculated topological order exists, use traverseInTopologicalOrder, else, use traverseDfs.
