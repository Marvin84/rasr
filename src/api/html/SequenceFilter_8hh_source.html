23 /**
24  * Filters input sequence according to a selection object.
25  * Input:
26  * default port: filtered stream
27  * selection port: vector-bool, i.e. sequence of bool values.
28  * Output:
29  * default port: selected features
30  *
31  * - A new selection is read, if the timestamp of the new data is not contained in the time interval
32  * of the current selection.
33  * - Data is filtered out, if the selection object contains 'false' at the position of the data.
34  * This implies as well, that selection has to have at least as many elements as data will come
35  * in its time interval.
36  */

38 private:

40  Flow::DataPtr<Flow::Vector<bool>> selection_;

42 private:
