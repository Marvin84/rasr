23 /*
24  * Cases:
25  * 1. Alphabets are identical (not just equal)
26  * => orignal automaton is returned
27  * 2. Alphabet mapping is identity (same set of symbols and same indices)
28  * => returned SlaveAutomaton has new alphabet but does not actually do anything
29  * 3. Mapping is partial identity (surjective) (same indices, but some symbols missing but)
30  * => only remove arcs from automaton
31  * (type changes; properties stay the same)
32  * 4. Mapping is partial => remove arcs from automaton and map indices
33  * (type/properties of automaton change)
34  * 5. Mapping is complete => only map indices
35  * (type/properties of automaton change)
36  */
