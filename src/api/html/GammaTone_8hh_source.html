55 /**
56  * Provides an auditory-based Gammatone filter bank. The filters are
57  * implemented as IIR filters in the time domain
58  *
59  * For details, see
60  * X. Zhang, M. G. Heinz, I. C. Bruce, L. H. Carney
61  * A phenomenological model for the responses of auditory-nerve fibers:
62  * I. Nonlinear tuning with compression and suppression
63  * JASA, February 2001 109(2), pp 648-670
64  * R. Schlueter, I. Bezrukov, H. Wagner, and H. Ney.
65  * Gammatone Features and Feature Combination for Large Vocabulary
66  * Speech Recognition.
67  * IEEE ICASSP, Honululu, HI, USA, April 2007.
68  *
69  * Center frequencies of the filters are calculated with the Greenwood
70  * function with human parameters:
71  * c = (165.4*10^2.1 * x -1), where x = proportion of basilar membrane length
72  * The filter bandwidth is calculated from the center frequency according to
73  * the Equivalent Rectangular Bandwidth measure:
74  * ERB(c) = 1.019 * 24.7 * ( (4.37*c)/1000 +1 )
75  *
76  * The reduction of temporal and spectral dimensionality of the filter bank
77  * outputs is performed by the Flow nodes TemporalIntegration and SpectralIntegration.
78  */

210  * Gammatone filter bank node
211  *
212  * Parameters:
213  * cascade: Order of the Gammatone filter.
214  * 4th or 3rd order filters best represent the response of
215  * human basilar membrane best
216  * minFreq, maxFreq: lower and upper frequency limits
217  * channels: number of the filter bank channels
218  * centerFrequencyMode: mode of center frequency distribution;
219  * human: greenwood function (with human parameters) based center frequency sampling are not in harmony with the filter bandwidth which based on ERB
