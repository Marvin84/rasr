Detailed Description

The HistoryManager serves as a memory manger for the HistoryDescriptors used by a LanguageModel.

Each LanguageModel creates an appropriate HistoryManager. Clients should never bother about this class. HistoryManager also plays the role of a virtual method table for HistoryHandle.
