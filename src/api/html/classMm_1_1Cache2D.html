Detailed Description

template<class O>
class Mm::Cache2D< O >

Two dimensional cache class optimized for short creation time.

Optmiziations: -isCalculated_ array is an optimized array with fast initilization time. -objects_ is no STL vector to avoid any calls to the constructor of the objects when after the memory.
