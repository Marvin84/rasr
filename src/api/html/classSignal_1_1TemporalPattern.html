Detailed Description

Estimation of temporal pattern (TRAP).

Implementation of the TRAP based feature extraction introduced in "TRAPs - Classifiers of Temporal Patterns" by H. Hermansky, S. Sharm ICSLP 1998.

The main idea is to use a large window of up to 50 frames for each dimension and apply a cosine transform for decorrelation and dimension reduction.
