Detailed Description

Basic event driven XML parser.

The XML Parser is built in top of libxml2 (the XML parser for Gnome). This class provides a thin wrapper around the SAX interface of libxml. All callback entries are converted into virtual member functions which can (and should) be overridden to implement event driven Xml parsers.

See also
full documentation of libxml: local or on the web
