Construct a prefix tree.

All sequences in the range [begin, end) are included in the prefix tree. The element at position end MUST exists although it is not used, since a sentinel is stored there during the construction phase.

Search prefixes of a string.

lookup() finds the longest sequence contained in the tree which is a prefix of seq and is not longer than maxLen or symbols. lookup() returns the range of items with this sequence. On return maxLen is adjusted to the length of this found sequence.

If no matching sequence is found, an empty range is returned and maxLen is set to -1. Note that this is different from the case of a zero length match, when the empty sequence is in the tree.

Parameters
seqsequence to search for
maxLenmaximum length of matching sequence

a pair of iterators designating the range of items having the longest prefixes
