Detailed Description

Compressed version of either Fsa automaton or OpenFst automaton.

The data is read from a memory mapped file. Limitations apply to the number of arcs per state (u16), number of epsilon arcs per state (u8) and number of labels (u16). Labels are stored in OpenFst format (Epsilon = 0)
