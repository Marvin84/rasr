Detailed Description

template<typename T>
class Nn::Prior< T >

Handles the class prior probabilities.

Priors can be read and written to file in Math::Vector formats. Alternatively, the priors can be estimated from a mixture set or from the class counts accumulated in a statistics object. A float parameter for scaling the prior can be specified.

In order to make old experiments reproducible, a compatibility mode is provided for estimating the priors from a mixture set.
