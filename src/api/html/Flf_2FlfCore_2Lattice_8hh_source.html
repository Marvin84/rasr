234  /**
235  * Each slot has the following properties
236  * 1) each word occurs at most once
237  * 2) the slot arcs are sorted by label ids, starting with the lowest label id
238  * 3) if posteriorId is valid, then
239  * sum over all scores[posteriorId] per slot equals 1.0,
240  * i.e. each slot describes a probability distribution over all words
