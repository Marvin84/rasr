const Core::ParameterBool Flow::paramVectorSequenceNodeReverse("reverse","if true, indices run backwards starting at the back of the inputs", false)

Vector sequence filter.

All input packets must be vectors, which will be put into a sequence of vectors.

Inputs: many (e.g extracted at same timestamp), dynamically generated. Outputs: the same, but on one stream. All inputs consume the same number of packets. The dimension of each output vector will be the same than each input vector.
