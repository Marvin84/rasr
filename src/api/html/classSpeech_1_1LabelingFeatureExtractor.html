Detailed Description

Assigns labels to feature vectors Extracts features and corresponding string labels from the underlying data source network.

-Label is assigned to a feature vector if label.startTime() <= feature.startTime and label.endTime() >= feature.endTime. -Feature extraction is interrupted if no label could be found for a feature vector.

The processAlignedFeature function of the LabeledFeatureProcessor objects is called with each feature and with the corresponding label index. Where the label-index is the position of the label (starting with zero) in the list of lables.
