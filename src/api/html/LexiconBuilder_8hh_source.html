28 /**
29  * Construction of the lexicon transducer.
30  * For each lemma pronunciation a separate path from the initial
31  * state to a final state is created.
32  * The lexicon is not closed, i.e. no transitions from final states
33  * to the initial state are created.
34  * The closure is added using close().
35  * Closure can be done using either only a epsilon transition or
36  * additional transitions for all lemma pronunciation with an empty
37  * syntactic token sequence (silence and noise).
38  *
39  * Options:
40  * add empty tokens:
41  * add lemma pronunciations with empty syntactic token
42  * sequence to the lexicon transducer.
43  * mark initial phones:
44  * add an offset to all input labels (phonemes) leaving the
45  * initial state. required for the context dependency transducer
46  * construction using across-word models.
47  * add word boundary phones:
48  * add input labels word phones at the initial and final position of a
49  * pronunciation
50  * add word disambiguators:
51  * add a transition after each pronunciation with
52  * input label = wordLabelOffset() + lemma pronunciation id.
53  * intended for adding "word" labels at the word end.
54  * disambiguate homophones:
55  * add disambiguators for homophones. not required if
56  * word disambiguators are used. disambiguators are in the range
57  * disambiguatorOffset() .. disambiguatorOffset() + nPhoneDisambiguators()
58  *
59  */
