Detailed Description

Phonology for context dependent phonemes.

The ContextPhonology allophone is the PhonemeInContext. A phoneme in context is defined by its underlying phoneme and its left and right context phonemes. Such units are commonly referred to as "n-phones" (e.g. triphone, quinphone,...).

The conversion from a phoneme graph to a context-dependent phoneme graph involves replication of vertices and edges according to the various contexts in which they occur. The current implementation proceeds as follows:

  1. Determine all left contexts (histories) of each vertex.
  2. Determine all right contexts (futures) of each vertex.
  3. For each phoneme graph vertex build the cartesian product of left and right contexts and create an allophone graph vertex for each context.
