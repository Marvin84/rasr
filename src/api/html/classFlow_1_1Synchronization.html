Detailed Description

Synchronization: ensures that the input stream has the same number of elements and the same start-times than "target-time" stream.

Algorithm: -remove all elements with a start-time that do not occure in the "target-time" stream. -Error messages: -there is an element in the "target-time" stream whose start-time do not occure in the input stream.
