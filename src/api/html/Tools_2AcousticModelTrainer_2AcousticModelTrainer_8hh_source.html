24 /**
25  * Actions:
26  *
27  * - dry: drives an aligner (given in *.aligner.type). Useful for generating and saving an alignment.
28  * - create-model-acceptors: activates the model-acceptor-cache object for each segment of the corpus
29  * - show-mixture-set: dumps mixture set in XML format
30  * - combine-mixture-sets: combines mixture sets given in old-mixture-set-template and old-mixture-set-ids
31  * - map-mixture-set: maps mixture set given mapping file
32  * - accumulate-mixture-set-text-dependent: mixtures set estimator accumulates features according to a given alignment
33  * - accumulate-mixture-set-ext-independent: mixtures set estimator accumulates features for set of acoustic conditions
34  * - accumulate-tdc-sum: decision tree sum file estimator accumulates features according to a given alignment
35  * - accumulate-discriminatively: mixtures set estimator discriminatively accumulates features according to a given (weighted) alignment
36  * - score-features: acoustic scores of a given alignment accumulated for set of acoustic conditions
37  * - estimate-scatter-matrices-text-dependent: estimates between and within class scatter matrices
38  * according to a given alignment
39  * - estimate-scatter-matrices-text-independent: estimates between and within class scatter matrices
40  * according to set of acoustic conditions
41  * - estimate-lda-text-dependent: creates projector matrix by LDA according to a given alignment
42  * - estimate-lda-text-independent: creates projector matrix by LDA according to set of acoustic conditions
43  * - estimate-lda: creates projector matrix by LDA based reading scatter matrices form files.
44  *
45  * - accumulate-weight-histogram: generates a histogram over all state weights in the alignment
46  * The target histogram file is set using the parameter weight-histogram-accumulator.histogram-file
47  *
48  * Corpus description: configured in the selection "corpus"
49  * Aligner: configured in the selection "aligner" @see Speech::Application
50  * Trainers: E.g.: accumulater, scorer etc are configured in the selection "trainer"
51  */
