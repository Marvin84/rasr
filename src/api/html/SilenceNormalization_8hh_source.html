23 /**
24  * This node normalizes the fraction of silence in the outgoing signal.
25  * The returned speech segments contain exactly the requested
26  * amount of silence.
27  *
28  * An EM-like algorithm is used to segment the silence from the non-silence,
29  * which expects that there is both silence and speech in the segment. Therefore
30  * data put into this node should already have been somehow segmented externally.
31  *
32  * The constraints are: At least 1% of the signal must be silence, and at least 1%
33  * must be speech (see the parameter initialization-fraction).
34  * */
