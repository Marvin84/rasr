Detailed Description

template<class _Key, class _Data, class _HashFunction, class _EqualKey>
class Core::MruObjectCacheList< _Key, _Data, _HashFunction, _EqualKey >

MruObjectCacheList: list of ObjectCacheItems implementing the most recently used strategy.

ObjectCacheItems are stored in a hash map by their keys. Order of ObjectCacheItems is implemented by a linked list of keys.

Each ObjectCacheItem points to its corresponding element in the linked list. If an element is accessed the corresponding element is moved (erase, push_front) to the beginning of the linked list. last() returns last key in the linked list.
