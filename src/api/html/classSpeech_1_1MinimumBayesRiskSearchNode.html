Detailed Description

Search the lattice for string with minimal Bayes risk.

The result will always be a linear FSA representing a path in the lattice.

void Speech::MinimumBayesRiskSearchNode::processWordLattice ( Lattice::ConstWordLatticeRef  lattice,
Bliss::SpeechSegment *  segment 
)
virtual

The actual processing of the word lattice where the search will be performed.

The lattice's Fsas will be linear combined each with weight 1.0 and the resulting total Fsa will be searched.

For the future, the Fsa to take should be made configurable!

Parameters
latticethe word lattice to be searched.
segmentthe speech segment, the lattice belongs to.

Assumption: fsa is an n-best list or lattice with evaluation tokens as input labels.
